# CoinMiner

## option to start the process

-z -> just do a becnhmark
```
./coinminer -z
```

-t ABC -> to give a specific trigram
```
./coinminer -t ABC
```
-P -> to print a pretty way
```
./coinminer -P
```
-m -> to print only a minimum coin find (with number of begin c)
```
./coinminer -m 3
```
